# ObjectSearch

AviUtl2 にオブジェクト検索ウィンドウを追加します。

beta23a にて動作確認済（古いバージョンでは動かない可能性があります）

# 導入方法

1. [リリースページ](https://github.com/zopty/ObjectSearch/releases/latest) から `ObjectSearch.aux2` をダウンロード
2. AviUtl2 の画面にD&D
3. 「このプラグイン・スクリプトを信頼して使用する」をクリック
4. メニューバーの `表示` 内に `Object Search` が追加されます

# 使い方
- エフェクトが一覧表示されます
- 検索バーに文字を打ち込むと、結果を絞り込んで表示します
- 検索結果をダブルクリックすると、タイムラインにフィルタオブジェクトとして追加されます
